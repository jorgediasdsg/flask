# flask

Treinamento de flask api

# Configurar ambiente

    ```shell
    # Criando ambiente virtual
    virtualenv venv

    # Ativando ambiente virtual
    source venv/bin/activate

    # Instalando dependências
    pip install flask
    pipl install sqlalchemy
    ```

# Rodando o projeto

    ```shell
    # Rodando o projeto
    python app.py
    ```
